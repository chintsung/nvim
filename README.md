## Overview
This is my config file for neovim.

## Quick Start
- In Unix-Like system, put `init.vim` in `~/.config/nvim`
- In Windows, put `init.vim` in `~\AppData\Local\nvim`

## Use Vundle to manage and install plugins
- Unix-Like system
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.config/nvim/bundle/Vundle.vim
```
- Windows
```
git clone https://github.com/VundleVim/Vundle.vim.git ~\AppData\Local\nvim\bundle\Vundle.vim
```

Launch vim and run :PluginInstall
